# 手機版遊戲架構設計方案（MVP 優先）

## Context

PT Game Trible 是 3x10 窄棋盤三消遊戲，已有 Web + Flutter 雙平台。目標是手機版做成有深度的休閒手遊，先上線驗證再逐步加深。

### 世界觀 & 主題
- **主題**：擬人化貓咪（穿衣服、有職業、拿武器，但外觀是貓）
- **世界觀**：現代都市，貓咪們有一個人類不知道的**秘密組織**
- **故事**：玩家是新加入秘密組織的菜鳥特工貓，在城市各區域執行任務、保護貓咪社會的和平
- **敵人**：城市中的威脅（野生動物、邪惡組織的貓、失控的機器等）
- **風格調性**：輕鬆幽默 + 帥氣，類似「間諜家家酒」但全是貓

### 關鍵決策
- **團隊**：1-2 人，必須極度精簡
- **TA**：先休閒再漸進（先吸引休閒玩家，後期用養成留住中度玩家）
- **策略**：MVP 先上線驗證，根據數據迭代
- **不做抽卡** → 吸血鬼倖存者式解鎖（成就+貨幣混合）
- **不做**：放置、即時PvP、公會、塔防
- **只保留三排模式**（3x10 棋盤）

### 風險提醒
1. 三消+RPG 市場飽和 → 差異化靠 3x10 窄棋盤 + 貓咪主題 + 充能選擇
2. 系統過多會拖垮開發 → MVP 嚴格控制在 7 個核心功能
3. 充能/捕捉機制未驗證 → MVP 先用簡化版，觀察數據再決定
4. 貓咪主題需要大量美術資源 → MVP 先用 5 個角色，風格統一即可

## A. 核心遊戲循環（MVP 版）

### 每一局
```
選擇角色隊伍(3人) → 消耗體力進入關卡 → 三消戰鬥 → 結算(金幣/EXP/素材)
```

### 每次 Session（5-15 分鐘）
```
登入 → 領每日任務 → 消耗體力打 3-5 關 → 升級角色 → 體力不夠退出
```

### 戰鬥機制（MVP 簡化版）
- 消除方塊 → **自動造成傷害**（主要傷害來源）
- 同時累積對應屬性貓咪的**技能能量**
- 能量滿 → 點擊角色頭像 → **施放特工技能**（大招）
- 敵人定期反擊（有倒數預告）
- 消滅敵人 → 任務完成結算

> 💡 MVP 先不做「捕捉/防禦/回復」分支選擇。
> 技能就是攻擊/防禦技能，和現有 SkillSystem 概念一致。
> 驗證後如果玩家覺得太單調，再加入行動選擇分支。

### 闖關故事（貓咪秘密組織）

**章節設定（現代都市各區域）**：

| 章節 | 地點 | 故事 | 敵人類型 |
|------|------|------|---------|
| 第 1 章 | 後巷區 | 新人特工訓練 + 首次任務 | 流浪鼠幫、野狗 |
| 第 2 章 | 商店街 | 調查可疑的寵物店 | 寵物店打手、陷阱 |
| 第 3 章 | 港口區 | 攔截走私貨物 | 碼頭幫派、海鷗Boss |
| 第 4 章 | 摩天大樓 | 潛入企業總部 | 保全系統、機器人 |
| 第 5 章 | 地下鐵 | 追蹤邪惡組織線索 | 暗影貓特工（敵對組織） |
| 第 6 章 | 秘密基地 | 最終決戰 | 邪惡組織首領 |

> 每章 10-15 關，MVP 先做前 3 章（共 30-45 關）。
> 對話以輕鬆幽默為主，穿插貓咪特有的可愛行為（打呼嚕、追逗貓棒等）。

## B. 角色系統（極簡版）

### 資料結構
```dart
class CatAgent {
  String id;
  String name;
  String codename;     // 代號（特工風格）
  String breed;        // 品種（美短、曼赤肯、暹羅...）
  String attribute;    // 5 屬性之一，對應 5 種方塊顏色
  String role;         // 職業（突擊手/防衛者/支援者/破壞者/潛行者）
  int rarity;          // 1-4 (N/R/SR/SSR)
  int level;           // 1-50 (MVP 上限)
  int atk, def, hp;
  Skill activeSkill;   // 1 個主動技能
  String passiveDesc;  // 被動效果
  UnlockCondition unlock;
}
```

### 屬性系統（5 屬性，剪刀石頭布式相剋）

```
三角相剋（A → 剋 B → 剋 C → 剋 A）：
  🔴 屬性A  →  剋  🟢 屬性B  →  剋  🔵 屬性C  →  剋  🔴 屬性A

互相剋制：
  🟡 屬性D  ⟷  🟣 屬性E

剋制倍率：被剋方受到 1.5x 傷害，剋制方受到 0.75x 傷害
同屬性：1.0x（正常傷害）
```

> 📝 屬性命名暫定，後續再調整風格。
> 核心機制確定：5 色方塊 = 5 屬性，3+2 相剋結構。

### MVP 角色（5 個貓咪特工）

| 角色 | 代號 | 品種 | 屬性 | 職業 | 稀有度 | 解鎖方式 | 技能 |
|------|------|------|------|------|--------|---------|------|
| 阿焰 | Agent Blaze | 橘貓 | 🔴 A | 突擊手 | N | 初始角色 | 爆裂爪擊：ATK*2 傷害 |
| 小波 | Agent Tide | 曼赤肯 | 🔵 C | 支援者 | N | 通關 1-3 | 水霧屏障：回復 20% HP + 延遲敵人 1 回合 |
| 大地 | Agent Terra | 美短 | 🟢 B | 防衛者 | R | 500 金幣 | 鋼鐵毛球：減傷 50%，持續 2 回合 |
| 閃光 | Agent Flash | 暹羅貓 | 🟡 D | 破壞者 | R | 通關 1-15 | 雷光爪：全體傷害 ATK*1.5 |
| 影子 | Agent Shadow | 黑貓 | 🟣 E | 潛行者 | SR | 第1章全三星 + 3,000 金幣 | 暗殺突襲：ATK*3（低HP敵人額外+50%） |

> 角色設定：擬人化貓咪，穿著現代特工服裝（風衣、護目鏡、通訊耳機等），
> 各有鮮明個性。橘貓阿焰是衝動熱血的新人，黑貓影子是沉默寡言的老手。

### 養成（MVP 只做等級）
- 消耗 EXP 素材升級（關卡掉落）
- 每級提升 ATK/DEF/HP
- 技能效果隨等級線性成長
- 不做裝備、不做天賦、不做突破

## D. 體力系統

```dart
class EnergySystem {
  int maxStamina = 60;        // MVP 上限較低（遊戲初期關卡少）
  int currentStamina;
  Duration recoveryRate = Duration(minutes: 8);  // 每 8 分鐘恢復 1 點
  DateTime lastRecoveryTime;

  // 每天自然恢復 ~180 點，可玩 15-20 關
  // 升級時全額恢復
}
```

| 消耗 | 體力 |
|------|------|
| 主線關卡 | 4-8 |
| 每日素材關 | 6 |

### 體力獲取
- 自然恢復（核心）
- 升級全恢復
- 每日任務獎勵（20 體力）
- 鑽石購買（每日限 2 次：30/60 鑽石）
- 觀看廣告恢復（可選，增加收入）

## I. 驗證方式

1. **戰鬥整合**：消除紅色 → 火屬性角色能量增加 → 技能施放 → 傷害正確
2. **體力計算**：消耗體力 → 時間流逝 → 恢復正確 → 超過上限不增加
3. **角色升級**：消耗 EXP 素材 → 等級提升 → 攻防血提升 → 戰鬥傷害增加
4. **每日任務**：完成條件 → 進度更新 → 獎勵領取 → 隔天重置
5. **IAP**：購買鑽石 → 到帳 → 商品正確 → 收據驗證
6. **數值平衡**：模擬免費玩家 7 天，確認能通關到第 20 關左右、角色等級 ~15
7. **手機適配**：iPhone SE / iPhone 15 Pro Max / 各種 Android 尺寸測試
