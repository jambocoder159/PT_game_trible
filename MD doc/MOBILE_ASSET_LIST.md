# 貓咪點心屋 — 手機遊戲版素材清單（for Flow / Nano Banana 生成）

## Context

根據 `MD doc/REBRANDING_PLAN.md` 的品牌重塑計畫，遊戲從「貓咪特工（間諜風）」轉型為「貓咪點心屋（溫馨甜點小鎮風）」。需要為全新的世界觀、角色、敵人、場景製作一份完整的 AI 生圖素材清單，供 Flow 平台的 Nano Banana 模型使用。

**風格定調**：2D Q版平面繪本風、溫暖大地色系（橘/棕/奶油黃/橄欖綠）、半擬人貓咪（四腳站立有表情配件）

**生圖限制**：Flow / Nano Banana 只能輸出 JPEG，無法產生透明背景。角色/圖示類使用**白色或純色背景**，角色立繪**減少背景元素**以利後期去背。

## 生圖注意事項（for Flow / Nano Banana）

1. **風格一致性**：所有角色類 prompt 結尾加上統一風格描述 `soft watercolor lineart, gentle pastel tones, chibi cat standing on four legs, simple themed clothing, clean pure white background, no scenery, children's book illustration style, consistent rounded soft shapes` 確保風格一致
2. **角色進化延伸**：每階進化要保留原始角色的核心特徵（品種、配件、顏色），只升級裝備華麗度
3. **背景處理**：Flow/Nano Banana 輸出為 JPEG，角色/圖示類使用 **白色或純色背景 + 極簡背景元素**，方便後期用去背工具處理；場景背景類直接使用完整場景
4. **手機直式**：背景圖統一 1080×1920 直式適配手機
5. **色彩參考**：方塊五色嚴格對應 — 橘紅`#E8723A`/綠`#6BAF5B`/藍`#5BA4CF`/粉`#F0B0C8`/紫`#9B7EC8`
6. **去背建議**：角色立繪生成後，可用 remove.bg 或 Photoshop 快速去背轉 PNG
