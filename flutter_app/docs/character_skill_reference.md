# 角色技能參考表

> 最後更新：2026-04-13

## 設計原則

技能為**合併施放**：戰鬥效果 + 棋盤效果同時觸發，無雙選對話框。

### 稀有度遞增規則

| 稀有度 | 棋盤效果範圍 | 策略定位 |
|--------|-------------|---------|
| **N** | 轉色 2-3 格 / 洗版 | 基礎能量補充，入門好懂 |
| **R** | 轉色 5-6 格 / 隨機消除 5-6 格 | 中等轉色或清場，可影響戰局 |
| **SR** | 轉色 8 格 / 整列消除 / 清除異常 / 全色轉換 | 改變局面，定義隊伍核心策略 |
