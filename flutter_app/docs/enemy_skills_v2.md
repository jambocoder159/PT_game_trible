# 敵人特殊技能系統 v2

> 最後更新：2026-04-09

## 設計原則

每個技能必須滿足：
1. **可預見** — 玩家能看到「即將發生什麼」
2. **可決策** — 至少有 2 種應對方式
3. **可反制** — 對應到至少 1 個角色能力或玩家操作
4. **有代價** — 反制成功有獎勵感，忽略有合理懲罰

## 技能池（共 8 種）

### 棋盤干擾類（3 種）

### 1. 障礙方塊 `obstacle`

| 項目 | 內容 |
|------|------|
| 效果 | 在棋盤上生成 2~4 個灰色障礙格，無法被移動、不參與配對 |
| 解除方式 | (1) 相鄰消除 2 次自動碎裂 (2) 角色技能的「消除格/行/列」直接清除 |
| 玩家反制 | 消除型技能（窯窯/糖霜/藍莓/焦糖/可頌/肉桂/蘇打）直接打掉 |
| 視覺 | 灰色方塊 + 裂痕動畫（被相鄰消 1 次後顯示裂紋） |
| 參數 | 格數 2~4、觸發頻率每 3~5 回合 |
| 引入章節 | Ch2 |

### 2. 倒數毒格 `poison`

| 項目 | 內容 |
|------|------|
| 效果 | 將 2~3 個方塊染毒，顯示倒數 3→2→1→爆，歸零時對隊伍造成固定傷害 |
| 解除方式 | (1) 在倒數歸零前消除該格（安全，不扣血） (2) 技能消除直接清除 |
| 不解除代價 | 每個毒格爆炸造成敵人 ATK x 50% 傷害 |
| 玩家反制 | 主動消除（優先排列）、消除型技能清理、轉色技能覆蓋（毒性消失變好格子）、治療角色事後補血 |
| 策略深度 | 玩家要決定：花步數去消毒格，還是集中打敵人搶在毒爆前擊殺？ |
| 參數 | 格數 2~3、倒數 2~3 回合、觸發頻率每 3~5 回合 |
| 引入章節 | Ch3 |

### 3. 弱化標記 `weaken`

| 項目 | 內容 |
|------|------|
| 效果 | 標記 3~5 個方塊，被標記的方塊消除時只產生 50% 傷害和能量 |
| 持續 | 2 回合後自動消失，或被轉色技能覆蓋時立即移除 |
| 玩家反制 | 轉色技能（小麥/薄荷/露露/奶昔/棉花糖/可可）覆蓋弱化格 |
| 策略深度 | 轉色角色從「附帶效果」升格為「主動反制工具」 |
| 參數 | 格數 3~5、持續 2 回合 |
| 引入章節 | Ch4 |

### 戰鬥強化類（3 種）

### 4. 護盾 `shield`

| 項目 | 內容 |
|------|------|
| 效果 | 獲得額外 HP 層（maxHP 的 20~40%），需先擊破護盾才能扣本體血 |
| 玩家反制 | (1) 高倍率單體技能(小麥/肉桂/可頌)暴力破盾 (2) 破防(Tier 5)降低護盾有效值 |
| 特殊互動 | 斬殺技能依據「本體 HP」判定，不受護盾影響 — 護盾碎了就能秒 |
| 參數 | 護盾量 20%~40% maxHP |
| 引入章節 | Ch1 Boss |

### 5. 蓄力重擊 `charge`

| 項目 | 內容 |
|------|------|
| 效果 | 敵人頭上顯示「蓄力中」，下一回合造成 ATK x 3 倍傷害 |
| 玩家反制 | (1) 護盾角色(抹抹/奶昔/布丁)開盾硬扛 (2) 延遲角色(露露/蘇打)推遲攻擊 (3) 搶在蓄力回合擊殺敵人 |
| 策略深度 | 看到蓄力 → 要放盾？放延遲？還是 all-in 輸出搶殺？三條路線 |
| 引入章節 | Ch3 |

### 6. 狂暴 `rage`

| 項目 | 內容 |
|------|------|
| 效果 | HP 低於 30% 時，ATK x2、攻擊間隔 -1 |
| 玩家反制 | (1) 斬殺角色(焦糖/可可) <30% 觸發 +50% 傷害，搶先帶走 (2) 護盾角色扛住狂暴期 |
| 策略深度 | 狂暴(30%) vs 斬殺(30%) = 同閾值競速，遊戲核心決策之一 |
| 引入章節 | Ch4 |

### 策略壓制類（2 種）

### 7. 屬性壓制 `aura`

| 項目 | 內容 |
|------|------|
| 效果 | 敵人開場自帶屬性光環，該屬性被克制的方塊消除傷害 -50%（不影響能量） |
| 範例 | 敵人屬性 A(日)開光環 → 玩家 B(葉) 色方塊傷害減半 |
| 玩家反制 | (1) 帶克制屬性角色（如壓制 B → 帶 A 角色） (2) 避開被壓制屬性 |
| 策略深度 | 強迫思考組隊，不能無腦帶最強角色通吃 |
| 重要限制 | 只影響傷害，不影響能量收集 — 不會造成死局 |
| 顯示位置 | 關卡選擇畫面即可見「本關壓制 X 屬性」 |
| 引入章節 | Ch5 |

### 8. 回血 `heal`

| 項目 | 內容 |
|------|------|
| 效果 | 每 N 回合回復 maxHP 的 X%（5~10%） |
| 玩家反制 | (1) DoT(Tier 3)持續傷害抵消回血 (2) 斬殺角色在低血收頭 (3) 集中火力不給回血機會 |
| 策略深度 | 鼓勵「集火策略」而非平均分散傷害 |
| 參數 | 回血量 5~10% maxHP、頻率每 3~5 回合 |
| 引入章節 | Ch3 |
